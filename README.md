wsend-link
==================

Share a link through wsend
# Installation

     git clone https://github.com/abemassry/wsend-link.git
     cd wsend-link/
     alias wsend-link="$PWD/wsend-link"

# Usage

   **Usage:**
   
     wsend-link <url>

# Dependencies
    
  [wsend](https://github.com/abemassry/wsend)

# Info

There's probably other services which serve the same purpose, but
I needed this tool and it worked for me.
